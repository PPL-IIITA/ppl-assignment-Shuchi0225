Deployment and Testing--

$ git clone https://github.com/PPL-IIITA/ppl-assignment-Shuchi0225

$ cd ppl-assignment-Shuchi0225/que7/build/classes

$ cd ppl-assignment-Shuchi0225/que7/src
$ javac *.java
$ cd ..
