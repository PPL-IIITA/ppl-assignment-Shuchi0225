Deployment and Testing--

$ git clone https://github.com/PPL-IIITA/ppl-assignment-Shuchi0225

$ cd ppl-assignment-Shuchi0225/que6/build/classes

$ cd ppl-assignment-Shuchi0225/que6/src
$ javac *.java
$ cd ..
